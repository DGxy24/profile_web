Tentang Saya

Lulusan S1 jurusan Teknik Informatika dari Universitas Methodist Indonesia. Memahami beberapa bahasa pemrograman seperti Java, PHP, Python, MySQL, CSS, dan JavaScript. Saya juga menguasai framework PHP yaitu Laravel. Selama masa studi, saya telah mengembangkan berbagai proyek yang berfokus pada pengembangan back-end menggunakan Laravel. Saya memiliki minat untuk berkarir di bidang IT (Information and Technology), khususnya di bidang Web Development dan Software Engineering.
